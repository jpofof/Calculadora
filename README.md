Projeto Funções JavaScript (JS-2)

Este repositório contém um projeto prático desenvolvido para exercitar a lógica de programação em JavaScript, focando na manipulação do DOM e em cálculos matemáticos aplicados.

##  Funcionalidades
* **Média Aritmética:** Calcula a média entre dois números inseridos.
* **Cálculo de Potência:** Realiza exponenciação através de entradas dinâmicas (prompt).
* **Verificação de Ano Bissexto:** Algoritmo que identifica se um ano é bissexto ou não.
* **Sistema de Pagamentos:** Calcula descontos progressivos baseados no valor total e na forma de pagamento (Pix, Débito ou Crédito).
* **Classificação de Notas:** Sistema que define se um aluno está aprovado, reprovado ou em recuperação.

##  Tecnologias Utilizadas
* HTML5 para a estrutura.
* JavaScript (ES6+) para a lógica e manipulação de elementos.
* CSS3 (Estrutura básica nativa).

##  Como executar
1. Clone o repositório: `git clone https://github.com/jpofof/Calculadora.git`
2. Abra o arquivo `index.html` em qualquer navegador.
